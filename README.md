# Smart To-Do Calendar

APTECH DSL 2026 Task 8: A web app to manage tasks on a calendar.

## Features
- Monthly calendar view
- Add/edit/delete tasks with title, description, priority, due time
- Task status (Pending/Completed)
- Filter tasks (All/Pending/Completed)
- Month navigation
- Responsive for mobile/desktop
- Local storage for persistence

Live Demo: https://shahmeer-sk23.github.io/smart-todo-calendar/

Built with HTML, CSS, JS.
